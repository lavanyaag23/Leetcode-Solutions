# 📚 Queue

This folder contains solutions to LeetCode problems related to **Queues** implemented in Java.

A queue is a fundamental data structure that follows the **First In, First Out (FIFO)** principle. It is commonly used in scheduling, buffering, breadth-first search (BFS), and simulation problems.
