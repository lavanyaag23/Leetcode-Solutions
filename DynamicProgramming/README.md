# Dynamic Programming

This folder contains solutions to LeetCode problems related to **Dynamic Programming (DP)** implemented in Java.

Dynamic Programming is a powerful technique used to solve complex problems by breaking them into simpler overlapping subproblems and storing results to avoid recomputation.

---

## 📚 Topics Covered

* 1D Dynamic Programming
* 2D Dynamic Programming
* Knapsack Variations
* Subsequence Problems
* Partition DP
* House Robber Pattern
* Optimization Problems
* State Transition Problems
* Circular DP

## 🎯 Learning Objectives

* Understand DP state and transitions

* Solve optimization-based problems efficiently
* Handle special cases like circular arrays
* Strengthen recursion + memoization thinking

---

## 💡 Key Concepts

* **State Definition:** Define what `dp[i]` represents
* **Recurrence Relation:** Build solution using previous states
* **Base Cases:** Handle smallest inputs carefully
* **Optimization:** Reduce space complexity where possible
