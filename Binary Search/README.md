# 🔍 Binary Search

This folder contains solutions to LeetCode problems related to **Binary Search** implemented in Java.

Binary Search is a highly efficient searching technique that works on sorted data by repeatedly dividing the search space in half. It is widely used for search problems, optimization problems, and searching on the answer.
